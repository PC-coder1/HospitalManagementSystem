Built a console-based Hospital Management System using Core Java and JDBC to manage patients, doctors, and appointments.

Key Features:

* Implemented **CRUD operations** for Patients and Doctors.
* Developed modules to **view doctors, view patients, validate records**, and **book appointments**.
* Appointment booking includes:

  * Patient & doctor ID validation
  * Checking **doctor availability** for a given date using SQL queries
* Connected application with **MySQL database** using JDBC.
* Used **PreparedStatement**, ResultSet, exception handling, and modular OOP-based class design (Patient, Doctor, Main Module).

Technologies Used:
Java, JDBC, MySQL, OOPs, Exception Handling, SQL.
